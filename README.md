# 📻 Serrão Digital

Website institucional desenvolvido para a **Rádio Serrão Digital**, com foco em apresentar a emissora, divulgar eventos, fortalecer a comunidade e proporcionar uma experiência moderna e responsiva aos ouvintes.

O projeto foi originalmente desenvolvido utilizando uma estrutura clássica baseada em Bootstrap e, posteriormente, passou por uma ampla refatoração, recebendo melhorias de estrutura, semântica, responsividade, organização do código, experiência do usuário (UX) e identidade visual.
